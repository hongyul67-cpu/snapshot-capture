# 📸 SnapShot — 화면 캡처 도구

간단하지만 깔끔한 Windows 화면 캡처 프로그램입니다.

## 실행

`dist\SnapShot.exe` 를 더블클릭하면 바로 실행됩니다. (설치 불필요, 단일 파일)

> 첫 실행 시 Windows SmartScreen 경고가 뜨면 **추가 정보 → 실행**을 누르세요.
> (서명되지 않은 개인 제작 exe라 나타나는 정상적인 경고입니다.)

## 기능

### 이미지 캡처 (4가지)
- **⬚ 영역 선택** — 드래그로 원하는 사각형 영역 캡처 (ESC 취소). 선택하는 동안 이 프로그램 창은 자동으로 숨겨져 캡처에 잡히지 않습니다.
- **🖥 전체 화면** — 모든 모니터를 한 장으로 캡처
- **🪟 활성 창** — 현재 맨 앞 창만 캡처
- **🖵 모니터별** — 드롭다운에서 고른 모니터 하나만 캡처

### 동영상 녹화 (MP4)
- 대상(전체화면 / 활성창 / **지정 영역** / 특정 모니터)과 FPS 선택 후 **녹화 시작 / 정지**
- 소리 없이 화면만 녹화됩니다.
- 녹화 중에는 촬영 영역 가장자리에 **빨간 테두리**가 표시됩니다(테두리는 영상 바깥이라 촬영본엔 안 잡힘).

### 영역 고정
- 「📍 영역 고정」으로 드래그해 정한 영역은 저장되어, **스크린샷·동영상·연속 캡처의 "지정 영역"에서 공통 재사용**됩니다.
- ⬚ 영역 선택으로 찍은 마지막 영역도 자동으로 고정 영역이 됩니다. 설정은 저장되어 다음 실행에도 유지됩니다.

### 미니(요약) 모드
- 헤더의 「🗕 작은 창」을 누르면 핵심 버튼만 있는 **작은 항상-위 창**으로 축소됩니다(⤢ 로 원래 크기 복원).

### 연속 캡처
- 지정한 **초마다 자동 캡처**를 반복합니다. 대상·간격 설정 가능.
- 대상: 전체화면 / 활성창 / 모니터별 / **지정 영역**. 「📍 영역 지정」 버튼으로 드래그해 정한 영역을 반복 캡처할 수 있습니다 (지정한 영역은 저장되어 다음 실행에도 유지).

### 캡처 피드백
- 캡처 순간 캡처 영역에 **흰색 플래시 모션**이 잠깐 나타납니다.

### 자동 저장 & 파일명 규칙
- **저장 폴더** 지정 (기본: `내 사진\SnapShot`) · 「📂 열기」로 저장 폴더를 탐색기에서 바로 열기
- **이름**을 지정하면 → `이름_001.png`, `이름_002.png` ... 순번으로 저장
  (폴더에 있는 기존 번호 다음부터 이어집니다)
- 이름을 비우면 → `capture_20260712_140534_395.png` 처럼 **날짜_시간(밀리초)** 로 저장되어 절대 겹치지 않습니다.
- 형식: PNG / JPG 선택

### 단축키 (전역 — 다른 창을 쓰는 중에도 동작)
| 기능 | 기본 단축키 |
|------|------|
| 영역 선택 | Ctrl + Shift + 1 |
| 전체 화면 | Ctrl + Shift + 2 |
| 활성 창 | Ctrl + Shift + 3 |
| 모니터별 | Ctrl + Shift + 4 |
| 녹화 시작/정지 | Ctrl + Shift + 5 |
| 연속 시작/정지 | Ctrl + Shift + 6 |

단축키 칸을 **클릭한 뒤 원하는 키 조합을 누르면** 바로 변경됩니다.

## 설정 저장 위치
`%APPDATA%\SnapShot\config.json` — 폴더/이름/형식/단축키 설정이 자동 저장됩니다.

## 소스에서 다시 빌드하려면
`build.bat` 더블클릭 (또는 아래 명령)

```
.venv\Scripts\activate
pyinstaller --noconfirm --onefile --windowed --name SnapShot ^
  --collect-binaries imageio_ffmpeg --collect-data imageio_ffmpeg capture_tool.py
```

빌드 결과: `dist\SnapShot.exe`

## 사용 기술
Python 3.14 · Tkinter(GUI) · mss(화면캡처) · Pillow(저장) · pynput(전역 단축키) · imageio-ffmpeg(MP4 인코딩, ffmpeg 내장) · PyInstaller(exe)
